# Enforcarnos en que sea capaz de ensamblarse a sí mismo.

Excusa narrativa que se me ocurre:

En la intersección entre:

- tareas para las que la industria todavía necesita personas (textiles, montaje
de piezas pequeñas, cableado...)

- tareas que sirven para que el robot se monte a sí mismo su crecimiento sea
exponencial.

hay:

- Poner tornillos pequeños (muy difícil actualmente)

- Pasar cableado por las piezas del robot.

- Descargar las impresiones 3D y poner cada una en su sitio para la cadena de
montaje.

- Proceso de calibración.


Esas son el factor limitante para que el robot pueda autoensablarse.

Podemos centrarnos en encontrar una forma de resolver estos puntos críticos.


PROS:

- La idea de que se monte a sí mismo está chula.

- Las tareas son de interés industrial.


CONTRAS:

- Es muy difícil. No es la tarea con mejor relación impresionante/esfuerzo.

- Tenemos poco tiempo para hacerlo funcionar.
