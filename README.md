# Hackathon Nvidia LeRobot 2025

Este repositorio es para que vayamos preparandonos para el hackathon.

(Así también practicamos a usar git en equipo)


## Estructura

- `education/`: Conceptos que es importante saber antes de ir, con recursos
para aprenderlos. Sentíos libres de ir actualizándolos en función de como de
útiles os vayan pareciendo.

- `ideas/`: Ideas de proyectos que podríamos llevar al hackathon. Vamos
recopilándolas y trabajándolas aquí.

- `information.md`: Información importante sobre el evento.
