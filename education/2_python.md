# Python

Todo lo que programemos será en python, así que es importante que lo llevemos
fresco.

Librerías importantes:

- Numpy. Librería para trabajar con arrays y operaciones matemáticas.

- Pytorch. La librería que se usa para hacer modelos de machine learning en
python. Probablemente no tengamos que entrar tanto al detalle durante el
hackathon, pero es importante saber lo que es.

- Huggingface. Librería que nos facilita el uso de modelos de machine learning
ya entrenados. También es el repositorio más grande de modelos preentrenados.

- IsaacGym. Librería de Nvidia para simulación robótica. Creo que valorarán
positivamente su uso, así que estaría bien echarle un ojo. Además nos sirva
para ir probando cosas en simulación antes de ir.
