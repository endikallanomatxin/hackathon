# System

El sistema operativo que usarán los ordenadores que nos van a dejar para
entrenar modelos dijeron que iba a ser Ubuntu.

Es importante que sepamos sobre:

- Linux. Filesystem, usuarios, permisos, procesos...

- Ubuntu. Instalación de aplicaciones con apt

- Terminal.

    - Qué es stdin, stdout, stderr

    - Comandos básicos: ls, cd, mv, cp, rm, cat, grep, find, ps, top, kill...

- SSH. Saber conectarse a un servidor remoto vía terminal.
